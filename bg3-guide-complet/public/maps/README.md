# Fonds de carte BG3

Placez ici les images de fond de carte pour chaque acte.

## Fichiers attendus

| Fichier           | Description          | Format recommandé |
|-------------------|----------------------|-------------------|
| `acte1-bg3.webp`  | Carte de l'Acte 1    | 2048×2048 WebP    |
| `acte2-bg3.webp`  | Carte de l'Acte 2    | 2048×2048 WebP    |
| `acte3-bg3.webp`  | Carte de l'Acte 3    | 2048×2048 WebP    |

## Notes

- Format recommandé : 2048×2048 pixels (carré)
- Les images doivent être en format WebP pour des performances optimales
- En l'absence de ces fichiers, la carte affiche le placeholder SVG (`acte1-placeholder.svg`, etc.)
- Les coordonnées des marqueurs sont en pixels dans un repère [0, 0] → [1024, 1024]
